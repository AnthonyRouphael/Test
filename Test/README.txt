Idea of the project:
--------------------------
This code creates a simple search box where users can search for specific terms within a predefined set of articles , and the 
matched terms are highlited by yellow in search results and there is a counter for the results that displays a message of the
number of search results we have.


How to use the project:
-----------------------------
-The project have two files (test.html) & (test.css).
-to see the result of the code  right click on test.html file in test folder ->open with ->google chrome.
